# Jira Clone tests

## Installation

Requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies.

```sh
npm i
```

## Run tests

Run all tests available via first command.
You can specify other browser to run tests as well.

```sh
npx cypress run
npx cypress run --browser=chrome
```
